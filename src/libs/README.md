# Library Installation

Please install the following libraries into this directory:

1. **Concord**: Clone https://github.com/Tjakka5/Concord into `libs/Concord`
2. **Baton**: Clone https://github.com/tesselode/baton into `libs/baton`
3. **Flux**: Clone https://github.com/rxi/flux into `libs/flux`
4. **HardonCollider**: Clone https://github.com/vrld/HC into `libs/HC`
5. **STALKER-X**: Clone https://github.com/adnzzzzZ/STALKER-X into `libs/stalker` (or `libs/STALKER-X`)
6. **Luis**: Clone https://github.com/tesselode/luis into `libs/luis`
7. **hump**: Clone https://github.com/vrld/hump into `libs/hump`
8. **cargo**: Clone https://github.com/bjornusc/cargo into `libs/cargo`

Ensure the main `.lua` file for each library is accessible (e.g., `libs/Concord/init.lua` or similar). You may need to adjust `main.lua` paths if the folders are named differently.
